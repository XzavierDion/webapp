• 浙江新能机电科技有限公司
    浙江新能机电科技有限公司座落在风景秀丽、交通便利的台州市黄岩城区,是一家专业从事新能源电机研发、生产、销售和技术服务及新能源电机驱动方案设计、开发的高科技企业。
    公司始建于1 9 9 0 年, 占地面积7 0 0 0 0 平方米, 拥有建筑面积8 0 0 0 0平方米的现代化生产基地,公司拥有国内顶尖的新能源汽车电机、控制技术及整体方案设计专家团队，公司职工800余名,专业技术人员100多名,公司拥有先进的制造设备、完善的质量管理体系和近30年的电机生产制造经验。本公司产品广泛应用于新能源汽车主驱动，空压机，转向泵，电空调及伺服传动。
    • 品质工程师
      岗位职责：
      1、做好品质部公司质量发展规划、工作计划、协助管理品质队伍；
      2、处理和解决客户反馈的质量问题，满足客户的质量需求，不断提高公司产品质量满意度；
      3、协助检查、督促公司日常品质部工作；
      4、负责制定公司各类检验标准和检验操作规定；
      5、确保对产品及完成品实施检查并作出判定和处置，保证产品质量；
      6、参与质量分析、编制质量控制计划，设计质量控制卡，确定质量控制点；
      7、组织开展质检方面相关业务学习、培训，提高品质检验、管理水平。
      任职要求：
      1、大专以上学历；
      2、机械、电机、控制类相关专业毕业；
      3、熟悉永磁同步电机制造工艺；
      4、有5年以上品质分析、品质控制、品质改善工作经验，能够根据品质数据，作出品质改善方案，并执行方案；
      5、工作认真负责，严谨细致，有较强的分析解决问题能力，良好的团队协作精神，服从工作安排。
    • 工艺工程师
      岗位职责：
      1、熟悉永磁电机结构和工艺；
      2、参与新产品设计工艺可行性评审工作；
      3、负责产品生产具体工艺流程的设计和优化；
      4、负责生产作业指导书、工艺守则等技术文件的编制、输出及优化；
      5、负责各生产过程工艺动作的精益分析，推进工艺技术文件的标准化工作；
      6、负责各类夹具、工装的设计、优化及相应图纸输出；
      7、负责各类工装、夹具的应用验证工作，并做好验证记录；
      8、负责所有工装委外制造过程的技术答疑及监管；
      9、负责落实工艺施工过程的现场指导、培训等工作；
      10、负责新工艺调研，工艺提升电机质量的研究。
      任职要求：
      1、机械设计制造及其自动化、电器工程及其自动化等相关专业，本科及以上学历；
      2、从事电机工艺工作3年及以上，有同行业知名电机企业任职工作经历；
      3、有批量电机生产工作经验优先；
      4、有永磁电机工艺工作经验优先；
      5、热爱电机工艺工作。
